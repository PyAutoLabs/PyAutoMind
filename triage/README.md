# triage/

Prompts whose work-type classification is not yet clear. Park a prompt here with
a short HTML-comment note at the top explaining what needs deciding, then move it
into the right `<work-type>/<target>/` folder once the kind of work is obvious.

This is a holding area, not a work type — PyAutoBrain does not route from here.
See [`../ROUTING.md`](../ROUTING.md) and the README "Prompt taxonomy" section.
